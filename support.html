How can we help?

Send bug reports, TestFlight requests, privacy questions, and account questions to MiloFlow support.

Email support Read privacy notes

Useful details

For bugs, include your device model, iOS version, selected provider, whether you were using voice or text, and what happened right before the issue.

Cloud API keys

Gemini Cloud, Gemini Live, and optional Gemini embeddings use your Google AI Studio API key. MiloFlow support cannot view or recover your API key.

Vault memory

You can refresh changed vault files, delete the local memory index, or forget the selected vault folder from MiloFlow settings.

Apple Watch

The watch companion is a Gemini Live remote. Talk from your watch while iPhone handles tools, memory, history, transcript, and audio output.
