Memory you can inspect

AI Memory is app-owned, not model-owned.

MiloFlow retrieves memory before the model answers. That keeps Local Gemma, Gemini Cloud, and Gemini Live using the same reviewable memory rules.

L

Local index

Search your selected Markdown vault with a local index that stays on this device.

G

Optional embeddings

Gemini embeddings can improve recall, but only after you opt in with your own API key.

V

Vault controls

Backfill readable Markdown, refresh changed files, delete the local index, or forget the folder.

I

Incognito boundary

Incognito skips persistence, vault sync, memory recall, archive, and reusable provider state.

Local-first by design

Your notes begin on your device.

MiloFlow stores source records in local SQLite. When you choose, it can mirror readable Markdown into an Obsidian vault and build a local search index for memory.
