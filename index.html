Two model paths

Choose private local mode or live cloud voice.

Start local when privacy and cost matter. Bring your own API key when you want realtime cloud voice and stronger online analysis.

Mode Best for Experience Cost
Private local Personal Milo chats and everyday reflection Voice and text run on your device when local assets are ready No cloud usage bill
Live cloud Realtime voice, images, and heavier workspace analysis More fluid live conversation with cloud AI Uses your own API key
