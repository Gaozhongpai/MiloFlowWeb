Simple plans

Milo is free. Workspaces are Pro.

MiloFlow charges for app features, not bundled cloud usage.

Free

$0

  • Regular Milo and Incognito Milo
  • Local Gemma when assets are installed
  • Gemini Cloud with your own API key
  • Local vault memory search
  • Milo reminders and Apple Reminders linking

Pro

$4.99 / month

  • Project, Journal, and Discussion
  • Gemini Live realtime voice
  • Talk to Milo from your Apple Watch
  • Workspace summaries and action items
  • Workspace export
  • Advanced live memory and tools

Pricing may vary by App Store region. Cloud model usage is billed by Google through the API key you provide.
